# i-moscow-backend

Backend-проект нашей команды 2 трека с хакатона [Moscow City Hack](https://moscityhack.innoagency.ru/#main)

Серверная часть представляет из себя REST API для взаимодействия с приложением. 

## Стек

* Go 1.16
* JWT
* Firebase
* MongoDB
